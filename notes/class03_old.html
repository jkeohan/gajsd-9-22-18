Data Types

Data Types

• In the machine, there is only data

• The goal of computing is to collect, manage, manipulate, and store this data

• To manipulate data, it's important to know the data type:

⇒ What the data represents

⇒ Possible values for that type

⇒ Operations that can be performed on the data

• ALWAYS KNOW YOUR DATA TYPE!

Data Types

• Numbers

42, 1337, 3.14

• Strings

"lots of kittens", 'I am 29 years old'

• Booleans

Either true or false

• Arrays

Collections of related data, stored in [], and comma-separated

['manhattan', 'the bronx', 'queens', 'brooklyn', 'staten island']

Data Types

• Undefined - A variable without a value


  var x = 2;
  x
  // 2

  var y;
  y
  // undefined

• Null - Used to indicate no value

var x = null;

• Note: undefined ≠ null (more info)

Variables

Variables

• Variables keep track of data in a program

• Used to manage a program's "state"

• The var keyword declares a variable assignment

var num = 1337;

• Stores this piece of data in memory under num

• Use the variable's name to "call", or retrieve, its value from memory

Variables

• Take care when naming variables!

• Names should reflect the data they describe

• Other naming conventions:

⇒ Names should start with a lowercase letter

⇒ Cannot use spaces; use hyphens or underscores instead

⇒ Better yet, use camelCase:


  var current_temp = 82; // NO!!!
  var current temp = 82; // DEFINITELY NO!!!

  var currentTemp = 72; // YES!!!
Variables: Reassignment

1. Create a variable, x, and assign it to 2


  var x = 2;

2. Perform some arithmetic on x


  x * 2
  // 4

3. What's x now equal to?


  x
  // 2
Variables: Reassignment

• You must reassign variables to change them


  var x = 2;
  // 2

  x * 2
  // 4

  x
  // 2

  // REASSIGN
  x = x * 2
  // 4

  x
  // 4

Arrays

Arrays

• Arrays are a list-like data structure of comma-separated values

• Useful for storing a sequence of related data

• Declare arrays using square brackets: [ ]


  var greatActors = ['Keanu Reeves', 'Nicholas Cage', 'Tom Cruise'];

• Element - an item inside the array

• Index - the numerical location of each element in the array

⇒ "0-indexed", start at position 0

Arrays

• JS provides multiple methods for array creation:

⇒ The new keyword


  var drinks = new Array('beer', 'coffee', 'water')

⇒ Using the array literal


  var drinks = ['beer', 'coffee', 'water']

• While similar, there are crucial differences; please use the array literal

Arrays

• Use [index] to retrieve values from an array

• Remember that arrays are 0-indexed


  var drinks = ['beer', 'coffee', 'water']

  drinks[0]
  // 'beer'

  drinks[0+1]
  // 'coffee'

  var x = drinks.length - 1;
  drinks[x]
  // 'water'

  drinks[4]
  // undefined
Arrays

• To add a value to END of an array, use .push()


  var drinks = ['beer', 'coffee', 'water']

  drinks.push('capri sun')
  // 4

  drinks
  // [ 'beer', 'coffee', 'water', 'capri sun' ]
Arrays

• To add a value to the BEGINNING, use .unshift()


  var drinks = ['beer', 'coffee', 'water']

  drinks.unshift('tea')
  // 4

  drinks
  // [ 'tea', 'beer', 'coffee', 'water']

Array Helper Methods

Array Helper Methods

JavaScript provides tools to make working with arrays easier:

⇒ .push() and .unshift()

⇒ .shift() and .pop()

⇒ .reverse()

⇒ .join()

Array Helper Methods

• First, let's create an array with some values:


  var message = [];

  message.push(8)
  // 1

  message.push('r', 'e', 'b', 'm', 'u')
  // 6

  message.push('n', 's', 'i', 'A', 'G', 'K')
  // 12

  message
  // [ 8, 'r', 'e', 'b', 'm', 'u', 'n', 's', 'i', 'A', 'G', 'K' ]
Array Helper Methods

• Use .pop() to remove, and return, the last value:


  message.pop()
  // 'K'

• Use .shift() to remove, and return, the first value:


  message.shift()
  // 8

• Use .unshift() adds values to the beginning:


  message.unshift(1)
  // 11
Array Helper Methods

• Use .reverse() to reverse an array


  message.reverse()
  // [ 'G', 'A', 'i', 's', 'n', 'u', 'm', 'b', 'e', 'r', 1 ]

• Use .join() to join all elements into a string


  message.join()
  // 'G,A,i,s,n,u,m,b,e,r,1'

  message.join(' ')
  // 'G A i s n u m b e r 1'

Note: .join() takes a separator as an argument

Array Iteration

Array Iteration

• Consider an array of Tweets; how to render items in this list for a user?

• Iterating through a list of elements, one at a time, is a very common and useful practice in programming

• We will be learning more about for loops next class:


  var boroughs = ['manhattan', 'the bronx', 'queens', 'brooklyn', 'staten island'];

  for (var i = 0; i < boroughs.length; i++) {
    var borough = boroughs[i]
    console.log(borough)
  }

  // 'manhattan'
  // 'the bronx'
  // 'queens'
  // ...
Array Iteration

• JS arrays have several iteration methods:

.forEach()

.every()

.some()

.filter()

.map()

• Most require a function to be supplied as an argument; this function will be applied to each item in the array, individually

Array Iteration

for


  var boroughs = ['manhattan', 'the bronx', 'queens', 'brooklyn', 'staten island'];

  for (var i = 0; i < boroughs.length; i++) {
    var borough = boroughs[i]
    console.log(borough)
  }

Array.forEach


  var boroughs = ['manhattan', 'the bronx', 'queens', 'brooklyn', 'staten island'];

  boroughs.forEach(function (borough) {
    console.log(borough)
  })
Array Iteration

Array.filter()

⇒ Creates a new array with all elements that pass the test implemented by the provided function


  var numbers = [1,2,3,4,5];

  var numbersLargerThanTwo = numbers.filter(function (num) {
    return num > 2
  })

  numbersLargerThanTwo
  // [ 3, 4, 5 ]
Array Iteration

Array.map()

⇒ Creates a new array with the results of calling a provided function on every element in this array


  var numbers = [1,2,3,4,5];

  var timesFive = numbers.map(function (num) {
    return num * 5
  })


  timesFive
  // [5, 10, 15, 20, 25]
