JSON: JavaScript's JSON Object

• Like the Math object, JavaScript has a lib for working with JSON: JSON

• JSON.stringify() ⇒ Transforms JavaScript to JSON


  var jsonObject = JSON.stringify({ name: 'chandler' });
  // "{"name":"chandler"}"

  typeof jsonObject
  // string

• JSON.parse() ⇒ Transforms JSON to JavaScript


  var javaScriptObject = JSON.parse(jsonObject);
  // { name: "chandler" }

  typeof javaScriptObject
  // "object"
