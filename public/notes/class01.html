Development Definitions

Web Development:

The process of building applications for the web (the browser)

Frontend Development:

The development of client (ex: browser) code (HTML, CSS, JS) - what the user sees and interacts with

Backend Development:

The development of server-side code that handles routing, business logic, and databases - the "stuff behind the scenes"

Variables

Variables

• Variables keep track of data in a program

• Variables are assigned using one of three keywords: var, let and const

Never use var

let can be reassigned and has more specific scope.

const cannot be reassigned a different value once assigned

Variables

• Take care when naming variables!

• Names should reflect the data they describe

• Other naming conventions:

⇒ Names should start with a lowercase letter

⇒ Cannot use spaces; use hyphens or underscores instead

⇒ Better yet, use camelCase:


  var current_temp = 82; // NO!!!
  var current temp = 82; // DEFINITELY NO!!!

  var currentTemp = 72; // YES!!!
Variables: Reassignment

1. Create a variable, x, and assign it to 2


  var x = 2;

2. Perform some arithmetic on x


  x * 2
  // 4

3. What's x now equal to?


  x
  // 2
Variables: Reassignment

• You must reassign variables to change them


  var x = 2;
  // 2

  x * 2
  // 4

  x
  // 2

  // REASSIGN
  x = x * 2
  // 4

  x
  // 4

Data Types

Data Types

• In the machine, there is only data

• The goal of computing is to collect, manage, manipulate, and store this data

• To manipulate data, it's important to know the data type:

⇒ What the data represents

⇒ Possible values for that type

⇒ Operations that can be performed on the data

• ALWAYS KNOW YOUR DATA TYPE!

Data Types

• Numbers

42, 1337, 3.14

• Strings

"lots of kittens", 'I am 29 years old'

• Booleans

Either true or false

• Arrays

Collections of related data, stored in [], and comma-separated

['manhattan', 'the bronx', 'queens', 'brooklyn', 'staten island']

Data Types

• Undefined - A variable without a value


  var x = 2;
  x
  // 2

  var y;
  y
  // undefined

• Null - Used to indicate no value

var x = null;

• Note: undefined ≠ null (more info)

Data Types

• Strong vs Weak typing

• JS is weakly typed - the compiler does not enforce typing:

• 5 * 'cat'

// => NaN

•'25' * 5

// => 125

•'25' + 5

// => "255"

• ALWAYS KNOW YOUR DATA TYPE!

Data Types

• Use the typeof operator to determine data type


  typeof 42
  // 'number'

  typeof 'cat'
  // 'string'

  typeof false
  // 'boolean'

  typeof [1,2,3]
  // 'object'

  typeof 'beer' === 'string'
  // true

  typeof 42 === 'string'
  // false
Arithmetic Operators

• Use the standard arithmetic operators to manipulate numbers


  1 + 2
  // 3

  2 - 5
  // -3

  5 / 2
  // 2.5

  4 * 10
  // 40
The Math Library

• Use the Math object to perform advanced operations

• In Node, enter Math. and hit tab to see the available operations 


  Math.sqrt(4)
  // 2

  Math.PI
  // 3.141592653589793

  Math.random()
  // a random* number between 0 and 1
