Conditional Statements: Logical Operators

=== equal to

Logical Operators: == vs ===

• == (loose equality); Compares value only


  > '2' == 2
  // true

  > 1 == true
  // true

• === (strict equality); Compares both value and type


  > '2' === 2
  // false (the string two is NOT equal to the number 2)

  > 1 === true
  // false (the number 1 is NOT equal to the boolean true)

• Always use === and !== when comparing equalities

Conditional Statements: Logical Operators

  var x = 10;

  if (x === 10) {
    alert('bananas');
  } else {
    alert('apples');
  }



  var x = 'cat';

  if (x !== 5) {
    alert('tacos');
  } else {
    alert('salads');
  }

Truthy and Falsy Values

Truth/Falsy Values

• In addition to explicit true and false values, JS values are inherently truthy or falsy

• Truthy: values that translate to true when evaluated in a boolean context

• Falsy: values that translate to false when evaluated in a boolean context

• Result of type coercion (e.g. a string to a boolean)

• Note: all values are truthy, except for a limited set of falsy values

Truth/Falsy Values

Falsy Values:

1) false

2) null

3) undefined

4) NaN

5) 0

6) "" (an empty string)

Truthy/Falsy Values

Truthy:


  var num = 10;

  if (num) {
    console.log('this is a TRUTHY value');
  } else {
    console.log('this is a FALSY value');
  }

  // 'this is a TRUTHY' value
Truthy/False Values

Falsy:


  var emptyStr = "";

  if (emptyStr) {
    console.log('this is a TRUTHY value');
  } else {
    console.log('this is a FALSY value');
  }

  // 'this is a FALSY' value
Truthy/Falsy Values

Remember: truthy/falsey values are not equal to true or false

• Compare these values using == and ===:


  false == 0
  // true

  false === 0
  // false (the boolean false is not equal to the number 0)

  true == 1
  // true

  true === 0
  // false (the boolean true is not equal to the number 1)

Boolean Logic

Boolean Logic

• Often, we will want to perform more complex logic

• Simple: If it's hot, wear a t-shirt


  if (temp === 'hot') {
    wear('t-shirt')
  }

• More Complex: If it's hot and raining, wear a rain jacket


  if (temp === 'hot' && weather === 'raining') {
    wear('rain jacket')
  }
Boolean Logic

• Three operators for more complex logic:

⇒ Binary operators that require two values

&& (AND)

|| (OR)

⇒ Unary operator that requires one value

! (NOT)

Boolean Logic: && (AND)

• The && operator requires both left and right values to be true


  true && true
  // true

• Any other combination will return false


  true && false
  // false

  false && true
  // false

  false && false
  // false
Boolean Logic: || (OR)

• The || operator requires just one of the left or right values to be true


  true || false
  // true

  false || true
  // true

• Only false || false will return false


  false || false
  // false
Boolean Logic: ! (NOT)

• The ! operator takes a value and returns the OPPOSITE boolean value


  !true
  // false

  !false
  // true

  !(true && true)
  // false

  !(true || false)
  // false

  // can also be used on truthy/falsy values
  !([].length)
  // true

Loops

• Loops allow us to repeat the same action a certain number of times

• The same block of code is executed at each step in the loop:


  for (var i = 1; i <= 5; i++) {
    console.log(i);
  }

  // 1
  // 2
  // 3
  // 4
  // 5

• JavaScript offers a variety of methods for looping

• We will be covering the for loop

Loops: for

• for loops run an explicity set number of times

• The following loop will print the numbers one through five:


  for (var i = 1; i <= 5; i++) {
    console.log(i);
  }

• initialization: declares the loop's counter var i = 1;

• condition: determines if the loop executes i <= 5;

• iteration: changes the counter after each loop i++

• body: code block that runs once per loop console.log(i)

Loops: for

• for loops are useful for iterating over arrays - similar to .forEach()


  var boroughs = ['manhattan', 'the bronx', 'queens', 'brooklyn', 'staten island'];

  for (var i = 0; i < boroughs.length; i++) {
    var boro = boroughs[i]; // ex: boroughs[0] would return 'manhattan'
    console.log(boro);
  }

• Be sure to initialize with your counter set to 0

• Run the loop for the duration of the array's length

• Access the array structure inside the loop using the current value of the counter
