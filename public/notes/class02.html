Arrays

Arrays

• Arrays are a list-like data structure of comma-separated values

• Useful for storing a sequence of related data

• Declare arrays using square brackets: [ ]


  var greatActors = ['Keanu Reeves', 'Nicholas Cage', 'Tom Cruise'];

• Element - an item inside the array

• Index - the numerical location of each element in the array

⇒ "0-indexed", start at position 0

Arrays

• JS provides multiple methods for array creation:

⇒ The new keyword


  var drinks = new Array('beer', 'coffee', 'water')

⇒ Using the array literal


  var drinks = ['beer', 'coffee', 'water']

• While similar, there are crucial differences; please use the array literal

Arrays

• Use [index] to retrieve values from an array

• Remember that arrays are 0-indexed


  var drinks = ['beer', 'coffee', 'water']

  drinks[0]
  // 'beer'

  drinks[0+1]
  // 'coffee'

  var x = drinks.length - 1;
  drinks[x]
  // 'water'

  drinks[4]
  // undefined
Arrays

• Use [index].indexOf() to retrieve an items position in the array.

• If found the index will be returned otherwise it will return -1

• It will return only the first instance found


  var drinks = ['beer', 'coffee', 'water']

  //found it
  drinks.indexOf("beer")
  0

  //didn't find it
  drinks.indexOf("wine")
  -1

• .indexOf() can be used to remove duplicates from an array, which will be useful to know for bonus question #2 in upcoming lab

Arrays

• To add a value to END of an array, use .push()


  var drinks = ['beer', 'coffee', 'water']

  drinks.push('capri sun')
  // 4

  drinks
  // [ 'beer', 'coffee', 'water', 'capri sun' ]
Arrays

• To add a value to the BEGINNING, use .unshift()


  var drinks = ['beer', 'coffee', 'water']

  drinks.unshift('tea')
  // 4

  drinks
  // [ 'tea', 'beer', 'coffee', 'water']

Array Helper Methods

Array Helper Methods

JavaScript provides tools to make working with arrays easier:

⇒ .push() and .unshift()

⇒ .shift() and .pop()

⇒ .reverse()

⇒ .join()

Array Helper Methods

• First, let's create an array with some values:


  var message = [];

  message.push(8)
  // 1

  message.push('r', 'e', 'b', 'm', 'u')
  // 6

  message.push('n', 's', 'i', 'A', 'G', 'K')
  // 12

  message
  // [ 8, 'r', 'e', 'b', 'm', 'u', 'n', 's', 'i', 'A', 'G', 'K' ]
Array Helper Methods

• Use .pop() to remove, and return, the last value:


  message.pop()
  // 'K'

• Use .shift() to remove, and return, the first value:


  message.shift()
  // 8

• Use .unshift() adds values to the beginning:


  message.unshift(1)
  // 11
Array Helper Methods

• Use .reverse() to reverse an array


  message.reverse()
  // [ 'G', 'A', 'i', 's', 'n', 'u', 'm', 'b', 'e', 'r', 1 ]

• Use .join() to join all elements into a string


  message.join()
  // 'G,A,i,s,n,u,m,b,e,r,1'

  message.join(' ')
  // 'G A i s n u m b e r 1'

Note: .join() takes a separator as an argument

Array Iteration

• JS arrays have several iteration methods:

.forEach()

.every()

.some()

.filter()

.map()

• Most require a function to be supplied as an argument; this function will be applied to each item in the array, individually

Array Iteration

for


  var boroughs = ['manhattan', 'the bronx', 'queens', 'brooklyn', 'staten island'];

  for (var i = 0; i < boroughs.length; i++) {
    var borough = boroughs[i]
    console.log(borough)
  }

Array.forEach


  var boroughs = ['manhattan', 'the bronx', 'queens', 'brooklyn', 'staten island'];

  boroughs.forEach(function (borough) {
    console.log(borough)
  })
Array Iteration

Array.filter()

⇒ Creates a new array with all elements that pass the test implemented by the provided function


  var numbers = [1,2,3,4,5];

  var numbersLargerThanTwo = numbers.filter(function (num) {
    return num > 2
  })

  numbersLargerThanTwo
  // [ 3, 4, 5 ]
Array Iteration

Array.map()

⇒ Creates a new array with the results of calling a provided function on every element in this array


  var numbers = [1,2,3,4,5];

  var timesFive = numbers.map(function (num) {
    return num * 5
  })


  timesFive
  // [5, 10, 15, 20, 25]
