Q: Looking at my codebase at work, I don't see `class App extends React`, rather const BankTransferDetails = ({ trade, inModal }) => {...} Is this a different way?

A: Yes...There are two ways to create components...As an ES6 Class and as a Functional Component.

Q: Can you use React, JS, and Jquery in the same JS file?

A: Yes...but you never will...

Q: It seems like there's a lot of set up in React to get basic things working- are there shortcuts or boilerplates for if you wanted to make something quickly?

A: Yes...There is a Node NPM package called "Create-React-App"

Why Use React?

• The answer is based on how we respond to the following question:

⇒ Are you building a application or just a basic html/css site?

⇒ Where is the data in your app located...in the DOM, JS or both?

⇒ Does the app need to keep track of any changes to the data?

⇒ Does changing the data effect more than one element in your code?

⇒ Are there any repeatable patterns that would benefit from using a base template?

⇒ Are you concerned about performance and scalability?
